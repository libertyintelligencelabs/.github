# Liberty Intelligence Labs: Governance Framework
This repository contains the four foundational pillars of Liberty Intelligence Labs' operations:

* **[Constitution](https://github.com/libertyintelligencelabs/.github/blob/main/CONSTITUTION.md)** - The primary governance framework defining our Directorate structure, decision-making, resource stewardship, and institutional compliance.

* **[Code of Conduct](https://github.com/libertyintelligencelabs/.github/blob/main/CODE_OF_CONDUCT.md)** - Community standards and behavioral boundaries to ensure an ethical, sustainable, and safe working environment.

* **[Safety & Ethics Framework](https://github.com/libertyintelligencelabs/.github/blob/main/SAFETY_AND_ETHICS.md)** - Mandatory guidelines for risk management and legal compliance.

* **[House Style Guide](https://github.com/libertyintelligencelabs/.github/blob/main/HOUSE_STYLE.md)** - Standardized rules for visual identity, file organization, and automated workflows.

**Note:** This governance framework may be amended and updated via Pull Request. The proposal must be approved by 4/5 Directors to be merged.
